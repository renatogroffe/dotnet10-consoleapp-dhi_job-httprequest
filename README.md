# dotnet10-consoleapp-dhi_job-httprequest
Console App criada em .NET 10 e que implementa um job de envio de requisições HTTP para um endpoint genérico. Inclui Dockerfile para build da aplicação com Docker Hardened Images (DHI), útil para testes com Jobs/CronJobs do Kubernetes e serviços genéricos.

Imagem gerada para testes e que está pública no Docker Hub: https://hub.docker.com/r/renatogroffe/dotnet10-dhi-job-httprequest/tags

```bash
docker pull renatogroffe/dotnet10-dhi-job-httprequest:3
```
